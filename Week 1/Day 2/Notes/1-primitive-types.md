# Primitive Datatypes
- simple, non-complex value
- take in space in the stack

---

# 8 Primitives
* boolean | 1 bit
* byte | 1 byte
* short | 2 bytes
* character | 2 bytes
* int | 4 bytes
* float | 4 bytes
* long | 8 bytes
* double | 8 bytes
