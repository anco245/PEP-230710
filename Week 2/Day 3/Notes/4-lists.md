# List
- data structure
- indexed like an array BUT is not fixed in size
- can contain duplicate values
- is part of the Collection API

---

# Nifty Methods
- add (just value or you can specify the index + value)
- isEmpty
- size
- remove (using index OR value)
- get (at the index)
- indexOf
- subList
- lastIndexOf
